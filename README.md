# Ant-sim
Langton's ant simulator with customisation and other nifty features

# Features
Hexagonal grids and tile grids with a maximum of 8 ants and 20 colours  
Presets so that you barely lift a finger  
Camera zoom that allows you to zoom and pan out to the glorious 400x400 grid
Fancy buttons and sliders that let you customise the ant and where it goes to your heart's extent!


# Controls  
Its simple! The only key that you ever have to press is "R", which takes you back to the main menu
